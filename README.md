## Snippets

#### Scale the Document's Fonts

```LaTeX
\usepackage{relscale}
\relscale{0.95} % or whatever scaling is desired
```

#### Set font for initials

```LaTeX
\usepackage{Carrickc,lettrine}
\renewcommand\LettrineFontHook{\Carrickcfamily}
```
